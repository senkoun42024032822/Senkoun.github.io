Mes projets

Projet actuel

Mon projet actuel consiste à concevoir et développer un site web personnel de type portfolio en utilisant les langages HTML et CSS. Ce site comporte plusieurs pages reliées par un menu de navigation commun et responsive. L’objectif de ce projet est de mettre en pratique les notions apprises en langage web.

Projet futur

Mon projet futur est de faire évoluer ce portfolio en y intégrant des fonctionnalités plus avancées. Je souhaite notamment ajouter de l’interactivité avec JavaScript et développer des applications web dynamiques.
